# **Динамический опрос**

*Простой и легкий веб-опросник, который загружает вопросы и варианты ответов из Json-файла, позволяет пользователю выбрать ответы и скачать их в формате `.txt` файла. Подходит для быстрого создания опросов, анкет и обратной связи.*

## Использование
1. Отредактируйте файл Json под свои вопросы. Пример:
```
[
  {
    "id": "q1",
    "type": "text",
    "text": "Как вас зовут и из какого вы города?"
  },
  {
    "id": "q2",
    "type": "single",
    "text": "Какая операционная система установлена на вашем компьютере?",
    "options": ["Windows", "macOS", "Linux", "Другая"]
  }
]
```
2. Откройте файл `index.HTML`  браузере, заполните опрос и нажмите кнопку " Скачать результаты(.txt) ".

## Разработка
### Требования
Для установки и запуска проекта необходим любой современный браузер. Дополнительных зависимостей  не требуется.

### Установка зависимостей
Проект не требует установки зависимостей - это чистый HTML,CSS,JS.

### Создание билда
Билд не требуется, так как проект полностью статический. Все файлы уже готовы к размещению.

## Тестирование
Для проверки работоспособности достаточно:
* Открыть файл `index.HTML`
* Убедиться, что вопросы подгружаются из файла `config.Json`
* Протестировать все типы вопросов (одиночный выбор, множественный, текст)

### Пример кода
```
РЕЗУЛЬТАТЫ ОПРОСА
========================================
Дата заполнения: 10.06.2026, 11:09:21
========================================

Вопрос 1: Как вас зовут и из какого вы города?
Ответ: Алина - Наро-Фоминск

Вопрос 2: Какая операционная система установлена на вашем компьютере?
Ответ: Windows

Вопрос 3: Какие технологии вы планируете использовать в работе? (Несколько вариантов)
Ответ: HTML / CSS, JavaScript / TypeScript, Python / Django
```

## FAQ

### Можно ли добавить больше типов вопросов?
Да. Сейчас поддерживаются single (одиночный выбор), multiple (множественный) и text (текстовое поле). Вы можете расширить код, добавив, например, тип rating или date.

### Как изменить название скачиваемого файла?
В файле `script.js` найдите строку `link.download: survey_results_${dateStr}.txt` и замените на нужное имя.

### Сохраняются ли ответы, если случайно закрыть страницу?
В текущей версии - нет. Если нужна эта функция, посмотрите раздел "To do" - там планируется добавить сохранение `localStorage`.

### Как добавить обязательные вопросы?
Сейчас валидация не встроена, но вы можете добавить ее самостоятельно: перед скачиванием проверять, что все ответы заполнены, и выводить предупреждение.
